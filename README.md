📌 About Diabetes Monitoring System

Diabetes Monitoring System is an intelligent healthcare platform designed to predict diabetes risk, monitor patient health, and provide AI-based recommendations for better diabetes management.

This system leverages Machine Learning models, health data analytics, and chatbot assistance to help users monitor and understand their diabetes risk effectively.
